Helix account setup:

Here is how to setup your helix account:

Requires:

  • NIH account
  • HPC account
  • NX client properly setup
    • (under persistent connections NX)

Instructions:

  1. start NX
  2. edit your .bashrc file
    • open a terminal
    • type nano .bashrc
  3. add the following lines to the end of the file 
    # Source ETPB definitions
if [ -f /data/MoodGroup/IRTA/etpb_bashrc ]; then
        . /data/MoodGroup/IRTA/etpb_bashrc
fi
  4. close the terminal and start a new one, you're all set!
